# transporte-seduc-frontend

Projeto frontend para a gestão de transporte da Secretaria de Educação. Aplicação web desenvolvida com foco em performance, tipagem estática e estilização utilitária.

## Tecnologias Utilizadas

O desenvolvimento do frontend deste projeto tem como base a seguinte stack:

* **[React](https://reactjs.org/)**
* **[Next.js](https://nextjs.org/)**
* **[TypeScript](https://www.typescriptlang.org/)**
* **[Tailwind CSS](https://tailwindcss.com/)**

## Como executar o projeto

### Pré-requisitos

Antes de começar, certifique-se de ter instalado em sua máquina o [Git](https://git-scm.com) e o [Node.js](https://nodejs.org/en/). 

### Clonando e Rodando a Aplicação

Abra o seu terminal e siga os passos abaixo:

```bash
# Clone este repositório
$ git clone https://github.com/adsifspcaragua/transporte-seduc-frontend.git

# Acesse a pasta do projeto no seu terminal
$ cd transporte-seduc-frontend

# Instale as dependências do projeto
$ npm install

# Inicie o servidor de desenvolvimento
$ npm run dev
```

Após a inicialização, a aplicação estará disponível para acesso em seu navegador poderá ser acessada no endereço: [http://localhost:3000](http://localhost:3000).
